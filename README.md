##  Description 
This project presents a complete exploratory data analysis (EDA) of the Superstore Sales dataset using Python. The goal is to uncover patterns in sales performance, identify high-performing categories and regions, understand discount behavior, and generate actionable business insights. The analysis includes data cleaning, outlier removal, visualizations, and final recommendations.

---

- Visualize important metrics using multiple charts
- Provide clear and practical business recommendations
